Task Launcher

Task Launcher will be automated software to open a browser and apps with a few clicks of a button.

The backend is written in Python utilizing Selenium for browser automation and Pywin for program automation.

My personal goal of this project is to gain more experience with writing automated software to become proficienct at task automation.

What is TonyLoweCodes.com?

This is my coding playground which is being built from the ground up as a way for me to develop new skills as well as showcase what I've already learned on my coding journey. It is quite literally a work it progress and will continue to evolve over time.

This site is hosted on AWS with Route 53's domain service. I felt it was important to go further than just building a site with a service provider that does much of the heavy lifting in the shadows.

I am writing all the code on VS Code and utilizing git for source version control. If you would like to view the source code for this site, feel free to visit my GitHub.

I will be progressively making new additions to this site as I experiment with new technologies and write new programs.

Task Launcher:

Task Launcher will be automated software to open a browser and apps with a few clicks of a button.

My personal goal of this project is to gain more experience with writing automated software to become proficienct at task automation.

The backend is written in Python utilizing Selenium for browser automation and Pywin for program automation.

My biggest struggle thus far was being unable to get selenium up and running with Microsoft Edge. I chalked it up due to my inexperience and then tried to automate a Chrome browser instead. To my surprise, the initial browser automation was successful. I then wanted to get a taste for automating the opening of apps which I did successfully using Pywinauto but that was extremely simple as no real coding is necessary to get Notepad to start in this fashion.

My next challenges will be opening multiple browser tabs and apps in one instance of running the Task Launcher.
